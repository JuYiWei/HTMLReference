Jetbrains 全家桶激活方式

MAC 实测：10.15.1，2019 款全家桶通过

1、先下载压缩包解压后得到 jetbrains-agent.jar，放在指定位置，不要变动
    /Users/xxx/MyAppConfig/JetbrainsConfig/jetbrains-agent.jar

2、启动 IDE

3、点击 "Configure" 或 "Help" -> "Edit Custom VM Options ..."
    如果提示是否要创建文件，请点"是|Yes"。

4、在打开的 vmoptions 编辑窗口末行添加 "-javaagent:/Users/xxx/MyAppConfig/JetbrainsConfig/jetbrains-agent.jar"

5、应用程序，显示包内容，打开 Contents/bin/xxx.vmoptions 进行编辑，内容同第三步

6、重启你的IDE。

7、点击 "Configure" 或 "Help" -> "Register..."
    选择License server方式，点击按钮 "Discover Server" 来自动填充地址，Activate 进行激活
